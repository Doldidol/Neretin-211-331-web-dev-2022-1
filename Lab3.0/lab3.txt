flex box - это модель с большим количеством свойств
 модель макета, которая позволяет упорядочивать элементы во flex-контейнере.
Родительский элемент - это flex-контейнер
дочерний (что находится внутри) - flex элементы
FLEX-DIRECTION
какая из осей (горизонтальная или вертикальная) будет основной зависит от flex-direction 
Оси - row - строка
	column - столбец

чтобы включить flex-контейнер нужно написать display: flex

justify-content - определяет выравнивание вдоль главной оси

align-items - определяет выравнивание вдоль поперечной оси

align-content - выравниваем линии в пределах flex контейнера, когда есть доп пространство на поперечной оси 
(подобно, как justify-content выравнивает элементы в пределах главной оси)

flex-basis - 

flex-grow - определяет способность расширяться при необходимости. 
Оно принимает значения от нуля, которое служит пропорцией.
Определяет сколько пространства внутри контейнера займет элемент.

flex-shrink - это свойство определяет способность гибкого элемента сжиматься при необходимости.

flex - ...


flex-wrap - По умолчанию гибкие элементы будут пытатьс уместиться на одной строке. Вы можете изменить это
и позволить элементам переходить на новую строку по мере необходимости с помощью этого свойства.

nowrap (по умолчанию): все flex элементы будут в одной строке

wrap: flex-элементы будут перенесены на несколько строк сверху вниз

wrap-reverse: flex-элементы будут перенесены на несколько строк снизу вверх
